Επεισόδιο #1 - Ολα είναι Τζόγος

Ποτέ δεν θα είσαι σίγουρος για τίποτα, κάθε απόφαση που παίρνεις εμπεριέχει αβεβαιότητα. Αυτό δεν σημαίνει πως είσαι στο έλεος της τύχης. Υπάρχουν εργαλεία, υπάρχει μέθοδος. Υπάρχει τρόπος να πάρεις σωστές αβέβαιες αποφάσεις, όσο όλοι γύρω σου ρίχνουν ζάρια και προσεύχονται.
Αν θες να μάθεις πώς να βλέπεις καθαρά εκεί που οι άλλοι βλέπουν τύχη, συνέχισε να διαβάζεις.
